About This TODO App

Welcome to our TODO application! This app is designed to help you manage your tasks efficiently and stay organized.

Features:

  • Add new tasks with a title and description.
  • View a list of all your tasks.
  • Edit existing tasks.
  • Delete tasks that are no longer needed.
  • Search for tasks based on title or description.

How to Use:

  1. Add a Task: Fill in the title and description in the form on the homepage and click "Submit".
  2. Edit a Task: Click the "Edit" button next to the task you want to modify.
  3. Delete a Task: Click the "Delete" button next to the task you want to remove.
  4. Search Tasks: Use the search bar in the navigation to find tasks by title or description.

Technology Stack:

This application is built using:

  • Flask - a lightweight web framework for Python
  • SQLAlchemy - an SQL toolkit and ORM
  • SQLite - a lightweight database
  • Bootstrap - a CSS framework for styling
